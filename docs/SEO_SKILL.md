# ACCDEE SEO Skill

Use this skill when improving SEO, share previews, crawler files, or content planning for ACCDEE.

## Goal

Make ACCDEE easier for search engines and Thai customers to understand without keyword spam or false promises.

SEO cannot be guaranteed at 100% because ranking depends on competition, content quality, backlinks, user behavior, site speed, and time. The safe goal is steady technical correctness and useful Thai content.

## Safe SEO Rules

- Do not promise national ranking.
- Do not stuff repeated keywords into the page.
- Do not hide text from users just for Google.
- Do not expose admin, wallet, orders, or private pages in sitemap.
- Do not index admin pages.
- Keep Open Graph and Twitter previews accurate.
- Use canonical URL: `https://www.accdee.shop/`
- Use Thai language metadata because the target market is Thailand.

## Required Technical SEO

The homepage should have:

- `<html lang="th">`
- clear `<title>`
- meta description
- canonical URL
- Open Graph title, description, image, URL, and type
- Twitter summary large image
- JSON-LD structured data
- crawler-safe `robots.txt`
- public `sitemap.xml`

Validation command:

```powershell
npm run check:seo
```

## Content Roadmap

Small safe content steps:

1. Add a short guide section or FAQ about choosing accounts for online marketing.
2. Add separate article pages later, for example:
   - `/guides/online-marketing.html`
   - `/guides/facebook-ads-account.html`
   - `/guides/business-manager.html`
3. Submit sitemap in Google Search Console.
4. Track real search queries before expanding more pages.

## Handoff

When changing SEO, report:

- Metadata changed.
- Sitemap/robots changed.
- Structured data changed.
- Validation result.
- Ranking claims avoided.
